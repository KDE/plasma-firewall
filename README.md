This is the repository for the Plasma 5 Firewall KCM.

# Requirements
- CoreAddons.
- KCMUtils.
- I18n.
- Plasma 5.18+.
- PlasmaQuick.
- Declarative.
- UFW or Firewalld.


# Select Backend Between UFW e Firewalld

To select firewalld backend when building the plasma-firewall, pass the parameter BUILD_FIREWALLD=1.
